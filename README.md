# jetbot_autoinstall script

Automated way to install pytorch, torchvision, torch2trt, trt-pose using a simple sh script

## Running the script

```
    cd auto_install_script
    chmod +x jetbot_auto_install.sh
    ./jetbot_auto_install.sh
```
